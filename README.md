# 📚 Library Management System

A full-stack Library Management System built with:
- **Frontend**: React.js (Vite)
- **Backend**: Python Flask REST API
- **Database**: MySQL

## Features
- Librarian login with password protection
- Add, Edit, Delete Books and Members
- Issue and Return books with fine calculation
- Users can search books and check availability
- View personally issued books with status

## How to Run

### Backend
```bash
cd backend
pip install flask flask-cors mysql-connector-python
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
